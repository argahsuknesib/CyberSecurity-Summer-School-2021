ΤΥΠΟΙ ΠΛΗΡΟΦΟΡΙΩΝ ΠΟΥ ΣΥΛΛΕΓΟΝΤΑΙ

 

Προκειμένου να σας παρέχουμε τις υπηρεσίες μας, θα σας ζητήσουμε να παρέχετε προσωπικές πληροφορίες που είναι αναγκαίες για την παροχή αυτών των υπηρεσιών προς εσάς. Εάν δεν παράσχετε τις προσωπικές σας πληροφορίες, ενδέχεται να μην είμαστε σε θέση να σας παρέχουμε τα προϊόντα ή τις υπηρεσίες μας.

 

Θα συλλέγουμε μόνο τις πληροφορίες που είναι αναγκαίες για τους προσδιορισμένους, ρητούς και νόμιμους σκοπούς και δεν θα τις επεξεργαζόμαστε περαιτέρω κατά τρόπο που δεν συνάδει με τους εν λόγω σκοπούς. Ενδέχεται να συλλέγουμε τους ακόλουθους τύπους πληροφοριών (που ενδεχομένως να είναι ή να μην είναι προσωπικές πληροφορίες):

·          Πληροφορίες συσκευής : δεδομένα σχετικά με τα υλικά (hardware), την απόδοση, την κατάσταση και τις ρυθμίσεις της συσκευής. Για παράδειγμα, το όνομα της συσκευής, η αναγνωριστική συσκευής (device id), οι ρυθμίσεις της συσκευής, η κατάσταση της συσκευής (online/offline), η διεύθυνση IP, η διεύθυνση MAC, το WIFI RSSI, ο δρομολογητής (router) SSID, η έκδοση υλικολογισμικού.

·          Πληροφορίες που μας παρέχετε ή που αποστέλλετε: Ενδέχεται να συλλέγουμε οποιαδήποτε προσωπική πληροφορία που μας παρέχετε, όπως τα στοιχεία του Mi Λογαριασμού σας.

·          Προσωρινά αποθηκευμένο βίντεο στο cloud : εάν έχει ενεργοποιηθεί η λειτουργία συναγερμού εντοπισμού κίνησης (motion detecting alarm function), ενδέχεται να συλλέγουμε ένα βίντεο κλιπ διάρκειας περίπου 10 δευτερολέπτων και να το μεταφορτώνουμε στο cloud όταν εντοπίζεται κάποια κίνηση. Κάθε βίντεο θα αποθηκεύεται για 7 ημέρες στον διακομιστή από προεπιλογή. Μπορείτε να απενεργοποιήσετε αυτήν τη λειτουργία καθ' οιανδήποτε στιγμή.

·          Δεδομένα τοπικού βίντεο : θα συλλέγουμε τα δεδομένα του βίντεό σας στην κάρτα αποθήκευσης που εισάγετε για λόγους προβολής ιστορικού.

·          Παροχή απομακρυσμένης πρόσβασης σε βίντεο πραγματικού χρόνου ή βίντεο αναπαραγωγής. Όταν παρακολουθείτε το βίντεο πραγματικού χρόνου ή το βίντεο αναπαραγωγής, θα δημιουργηθεί ένα κανάλι p2p για τη μετάδοση της ροής βίντεο. Όλα τα δεδομένα ροής είναι κρυπτογραφημένα κατά τη μεταφορά και δεν αποθηκεύονται στον διακομιστή.
